# Operating principles

How the People function runs when agents do the work and humans own the decisions.

1. **Agents recommend; humans decide.** Every consequential action is gated by an entitled
   human approval ([governance/hitl-matrix](../../governance/hitl-matrix.md)).
2. **Cite, don't paraphrase.** Answers about policy come from the canonical `policy/` notes,
   with the source and `last-reviewed` date attached. Edge cases escalate.
3. **One source per fact.** Knowledge → this vault; decisions/approvals → the event ledger;
   employee/candidate data → the HRIS/ATS. Nothing is duplicated.
4. **Everything is observable.** Agents work in public channels; every decision is one row in
   an auditable ledger.
5. **Fail closed.** Missing, stale, or untrusted inputs stop the work and escalate — they
   never get guessed.
